# 核心贡献：三个物理维度的突破

## 1. 从“经验工程”到“信息论极值”：提出  H = Epiplexity  定理

过去，我们只知道：

> “让模型多说几句废话（Let's think step by step）”能提升推理能力。

但问题在于：

- 废话要说多少？
- 怎么说？
- 怎样才算“刚刚好”？

全靠启发式调参。

论文提出了一个严格的等式：Entropy (H) = Epiplexity


其中：

- **Entropy（熵）**：序列中不可预测的随机信息量  
- **Epiplexity（表象复杂度）**：为解决问题必须经历的不可压缩逻辑步骤数量  

### 关键洞察

- 自然语言的 `H >> Epiplexity`  
  废话太多，信息稀释严重。
- 人造数据常常 `H << Epiplexity`  
  过于规律，模型死记硬背位置编码。

### 全息数据的核心准则

> 每一个 Token 的不可预测性，必须全部来自逻辑推演本身。

没有格式填充  
没有模板噪音  
没有“思考中...”式的浪费 Token  

这种“纯逻辑高密度流形”，才是保证 Scaling Law 不崩塌的物理前提。
